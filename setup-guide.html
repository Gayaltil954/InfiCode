Follow these steps to configure EmailJS for your InfiCode Solution website's contact form.

Step 1: Create EmailJS Account

  1. Go to EmailJS.com
  2. Click "Sign Up" and create a free account
  3. Verify your email address

Step 2: Add Email Service

  1. Log in to your EmailJS dashboard
  2. Click "Email Services" in the left sidebar
  3. Click "Add New Service"
  4. Choose your email provider (Gmail, Outlook, etc.)
  5. Follow the connection instructions
  6. Copy the Service ID (e.g., service_xxxxxxx)

Step 3: Create Email Template

  1. Click "Email Templates" in the left sidebar
  2. Click "Create New Template"
  3. Use the following template structure:

Subject:

New Contact Form Submission: {{subject}}

Content:

Hello {{to_name}},

You have received a new message from your website contact form.

Name: {{from_name}}
Email: {{from_email}}
Subject: {{subject}}

Message:
{{message}}

---
This email was sent from your InfiCode Solution website contact form.
  1. Save the template
  2. Copy the Template ID (e.g., template_xxxxxxx)

Step 4: Get Public Key

  1. Click "Account" in the left sidebar
  2. Find the "Public Key" section
  3. Copy your Public Key (e.g., xxxxxxxxxxxxx)

Step 5: Update Configuration

Open src/config.ts and update the EmailJS configuration:

export const CONFIG = {
  emailJS: {
    serviceId: 'service_xxxxxxx',    // Your Service ID
    templateId: 'template_xxxxxxx',   // Your Template ID
    publicKey: 'xxxxxxxxxxxxx'        // Your Public Key
  },
  // ... rest of config
}

  1. Build your TypeScript files: npm run build

  • No email received: Verify Service ID, Template ID, and Public Key
  • Error 400: Check template variable names match exactly
  • Error 403: Verify your Public Key is correct
